# HomeBud

**HomeBud** is a web app that helps households manage groceries, recipes, and shared expenses more efficiently.

## Core Features

- **Receipt scanning** for automatic grocery entries
- **AI recipe generation**
- **Household expense tracking** for multiple users

## About

HomeBud brings grocery management, recipe support, and shared expense tracking into one app.  
It is built for households that want a more organized and collaborative way to manage everyday tasks.

## Deployment

HomeBud can be self-hosted.

### Deploy with Docker

The app can be fully deployed in a **Docker container**, which makes installation and hosting much easier in different environments.

### Use with Nginx

For production deployment, HomeBud can be run behind **Nginx**.  
The Nginx configuration can be used as a reverse proxy in front of the Docker container to manage traffic and domain access.

### Requirements

To deploy HomeBud yourself, you generally need:

- Docker
- access to the project source code
- an Nginx configuration
- a server or VPS
